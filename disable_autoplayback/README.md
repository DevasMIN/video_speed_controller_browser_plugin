# YouTube Autoplay Blocker

Chrome-расширение для блокировки автоматического воспроизведения видео на YouTube при переключении на вкладку.

## Проблема

Когда открыто много вкладок YouTube, при переключении фокуса на одну из них видео автоматически начинает воспроизводиться. Это может быть раздражающим и неожиданным.

## Решение

Это расширение автоматически блокирует воспроизведение видео при переключении на вкладку YouTube, предоставляя вам полный контроль над тем, когда начинать просмотр.

## Установка

1. Откройте Chrome и перейдите в `chrome://extensions/`
2. Включите "Режим разработчика" (Developer mode) в правом верхнем углу
3. Нажмите "Загрузить распакованное расширение" (Load unpacked)
4. Выберите папку `disable_autoplayback`

## Функциональность

- ✅ Блокирует автоматическое воспроизведение при переключении на вкладку
- ✅ Удаляет атрибут autoplay с видео элементов
- ✅ Перехватывает попытки программного запуска воспроизведения
- ✅ Возможность включения/выключения расширения
- ✅ Работает на всех страницах YouTube
- ✅ Красивый интерфейс управления

## Как работает

Расширение использует несколько методов для блокировки автовоспроизведения:

1. **Отслеживание видимости вкладки** - определяет момент переключения на вкладку
2. **Запоминание состояния видео** - сохраняет какие видео играли ДО переключения на другую вкладку
3. **Умная блокировка** - останавливает только те видео, которые начали воспроизводиться автоматически при возврате на вкладку
4. **Перехват play() метода** - блокирует программные вызовы воспроизведения (только для автоматических)
5. **Удаление autoplay атрибута** - предотвращает HTML5 autoplay
6. **MutationObserver** - отслеживает динамически добавленные видео элементы

**Важно:** Если видео уже воспроизводилось до переключения на другую вкладку, оно продолжит воспроизводиться при возврате. Блокируется только НОВОЕ автоматическое воспроизведение.

## Использование

1. Установите расширение
2. Откройте несколько вкладок YouTube
3. Переключайтесь между вкладками - видео не будет воспроизводиться автоматически
4. Кликните на иконку расширения для настроек
5. Используйте переключатель для включения/выключения функциональности

## Настройки

- **Включить/Выключить** - быстрое переключение работы расширения
- **Сброс настроек** - возврат к значениям по умолчанию

## Иконка

Для работы расширения необходима иконка. Создайте файл `icons/icon.png` размером 128x128 пикселей или используйте любое изображение с символом паузы/стоп.

## Техническая информация

- **Manifest Version**: 3
- **Permissions**: storage
- **Host Permissions**: *://*.youtube.com/*
- **Content Scripts**: Запускается на всех страницах YouTube

## Примечания

- Расширение работает только на YouTube
- Не влияет на ручное воспроизведение (когда вы сами нажимаете play)
- Минимальное влияние на производительность
- Не собирает никаких данных
